Projeto conversor de moedas Java e Orientação a Objetos T6 - ONE
conversor de moedas utilizando API com 6 opções de conversão
Aprendendo a manipular Json e API. Projeto para ser executado
no própio terminal utilizando JAVA e biblioteca Gson.

Funcionalidades:

* Real para dólar
* Dólar para real
* Euro para peso argentino
* Euro para real
* Peso argentino para dolar
* Peso colombiano para euro
